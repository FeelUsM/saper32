# saper32
сапер, который я писал на 1м курсе

компилировать в Delphi 7, есть скомпилированный exe-шник

В обычном сапёре идёт "автоматика": автоматическое открытие клеток вокруг данной клетки, если вокруг данной клетки 0 мин.
Я сделал более умную "автоматику": автоматическое открытие клеток вокруг данной клетки, если число флагов вокруг данной клетки совпадает с количеством мин вокруг (с цифрой) данной клетки
и автоматическое проставление флагов вокруг данной клетки, если количество неоткрытых клеток вокруг данной клетки совпадает с количеством мин вокруг (с цифрой) данной клетки.
